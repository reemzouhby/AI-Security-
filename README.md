# Markdown Cheatsheet

## Headings
# H1
## H2
### H3
#### H4
##### H5
###### H6

## Text Formatting
**Bold** → `**Bold**`  
*Italic* → `*Italic*`  
***Bold + Italic*** → `***Bold + Italic***`  
~~Strikethrough~~ → `~~Strikethrough~~`  
`Inline code` → `` `Inline code` ``

## Blockquotes
> This is a blockquote  
> - Nested list inside blockquote

## Lists
### Unordered
- Item 1
- Item 2
  - Sub-item

### Ordered
1. First
2. Second
   1. Sub-item

### Task Lists
- [x] Completed
- [ ] Pending

## Links
